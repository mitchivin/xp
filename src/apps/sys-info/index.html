Developer Notes

This project recreates the Windows XP desktop using only HTML, CSS, and JavaScript. The goal was to capture the look and feel of XP, including the Start Menu, taskbar, and window behavior.

It began as an experiment to see what someone with little coding experience could build with the help of AI coding agents. All work was guided and reviewed by me, using AI as a tool for learning and creation. No external frameworks were used.
